# SMU First-Time Entering Student Profile and Risk Prediction Dashboard

## Purpose

This dashboard profiles first-time entering students using the SMU Biographical Questionnaire and identifies factors associated with Academic Achievement.

## Dashboard Outputs

### Student Profile

- Demographic profile
- Frequency tables
- Percentage distributions
- Home background profile
- Socio-economic profile
- Academic pressures
- Social pressures
- Emotional pressures

### Student Success Analytics

- Academic Achievement Classification
- Variable Importance Analysis
- Student Segmentation
- At-Risk Student Identification

## Installation

pip install -r requirements.txt

## Run

streamlit run app.py

## Data

Upload the SMU Biographical Questionnaire Excel dataset.
